# My links

A minimalist, static website designed to showcase your personal links (websites, social profiles, etc.) in a clean and professional layout. Perfect for replacing traditional "Linktree" services with a simple, customizable solution.

---

## Installation

This project requires only **two files** to be deployed via FTP. No server-side dependencies or configuration needed.

### Steps:
1. **Download the files**  
   - `index.html` (main page)  
   - `styles.css` (styling)  

2. **Connect to your FTP server**  
   Use your FTP client (e.g., FileZilla) to access your web hosting account.

3. **Upload the files**  
   Place both files in the **root directory** of your website (e.g., `public_html/` or `www/`).

4. **Access your site**  
   Visit your domain (e.g., `https://votre-domaine.com`).

---

### Customization  
- Edit the links in `index.html` (replace `href` values with your actual URLs).  
- Modify colors/fonts in `styles.css` if needed.  

> ⚠️ **Note**: This is a static site. No backend or server configuration is required.  
> 🔗 Example: `index.html` contains placeholder links (e.g., GitHub, LinkedIn). Update these to match your content.
